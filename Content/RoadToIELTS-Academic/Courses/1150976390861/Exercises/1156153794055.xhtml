What will I learn in this unit?

Static charts 1

You will ...

  1. 1. read a report and complete it with the missing information.
  2. 2. match different kinds of charts that show the same information.
  3. 3. listen and match percentages, fractions and ratios to charts.
  4. 4. identify verbs in the present simple in a report.
  5. 5. complete a report with verbs in the past simple.
  6. 6. write a report based on a static chart.
  7. 7. correct mistakes in a sample report.
