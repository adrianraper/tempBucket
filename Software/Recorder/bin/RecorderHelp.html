What does the Recorder do?
 
Recording
The main function of the Recorder is to capture what you say into a microphone. Click the Record button and start talking. You will see the recording appear as waveform as you talk. Once you fill up the screen it will automatically keep moving right. When you have finished talking click Stop. If you click Record again, it will add what you say next to what you already have. If you want to make a new recording, click New, then the Record button.
   
Saving
Once you have a recording that you want to keep, click the Save button. This will convert your recording into an MP3 file which you can then save onto your computer.
   
Playing
Click the Play button to listen to the current recording. You can click Pause to briefly stop the recording and then click Pause or Play to start from where you stopped. If you want to play from the beginning, click Stop and then Play.
   
Recording level
Even when you are not recording, the recording level meter on the right will show you how loudly the microphone is picking up your voice. Ideally the green level should be at least half way up, but not ever going red. You can use your computer's audio settings to change your microphone level. The slider next to the waveform changes the way you see the waveform, but it doesn't actually change the volume.
   
Comparison When used with other Clarity programs, the Recorder can display the waveform of a model audio in the bottom half of the screen. You can then compare this with a recording you make of the same speech to help you with your pronunciation, stress and rhythm. At the moment you can't directly open an audio file to use in this comparison, but this will be released in a future update.

Using it with other programs
 
Internet

Most of Clarity's programs include the Recorder. When you start a Clarity program, it checks to see if the Recorder is already running. If it isn't, then you will see a Recorder button when you open an exercise. If you click this you will get a popup window to start the Recorder. Once it is running, you will see the Record, Play, Stop, Save and Compare buttons in the program. You can use these to control the Recorder. They work just as if you clicked on the buttons in the Recorder itself. At any point, you can switch to look at the Recorder directly.

The Save button is not active until you have made a recording. The Compare button is not active until you have played an audio file that can be used as a model.
 
Networks
If you are running a Clarity program on a network, the program will only have the Record, Play and Stop buttons. If you want to save you need to switch to the Recorder and do it from there. Some older programs will not start the Recorder when they are launched - you will need to start the Recorder manually.
 
Removing the old Recorder

If you have installed or run a desktop or network version of Clarity programs, you will have also installed an old version of the Clarity Recorder. You can uninstall it from the Control Panel or from the C:\Clarity\Recorder\Remove.exe program. But if you run Clarity programs again, it may try to reinstall the old Recorder. To prevent this from happening, download and run this small patch. This puts a fake version of the old Recorder onto your computer that will stop any re-installation.

Any existing Clarity programs that you have running on the computer or network will work with the new Clarity Recorder, although you will have to remember to start it manually as it can no longer run automatically.

Performance
 
The Clarity Recorder doesn't have a time limit on how long you can record your voice. However, it uses up memory in your computer as you are speaking, so if you plan on recording for an hour, run a test first to make sure that you have enough memory space to do that!
