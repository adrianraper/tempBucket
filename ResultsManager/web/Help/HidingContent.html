Hidden content

You can hide parts of the content from members of your group. You might want to only let your Class B learners access the Lower Intermediate level of Tense Buster, for instance. So you would hide the other four levels from the group Class B.

How to do it

Click on the group that you want to hide content for. You can't hide content for an individual learner, just for a group. Once a group is clicked, the content tree on the left will show ticks next to all the parts of the content. Click on a tick to remove it, which will hide the content. You can click on a title, a course, a unit or an exercise.

If you click a course, for instance, it will hide or show all the units and exercises in that course.

What if you have a hierarchy of groups?

For example, you have a group called Year 1, with groups in it called Class A, Class B and Class C. If you click Year 1 and then hide Tense Buster Advanced, none of the learners in any of the groups will be able to see this course.

If you then click on Class C and hide Tense Buster Upper Intermediate, the Class C learners will also blocked from this course. But Class A and Class B learners will still be able to see Upper Intermediate.

Clearing all hidden content

If you want to reset everything so that all learners can access everything, select the top level group and make sure that all titles are ticked. You should see a solid black tick, which shows that everything within that section is ticked. If you see a grey tick, it means only some of the section is ticked.
